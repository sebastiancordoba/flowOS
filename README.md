# flowOS

Calentamiento cognitivo matutino: en lugar de abrir redes sociales al despertar,
5–10 minutos de ejercicios mentales con dificultad adaptativa.

**Ejercicios:** cálculo rápido · Stroop · n-atrás · atención sostenida (SART) ·
cambio de tarea · respiración guiada.

Sin cuentas, sin backend: todo se guarda en tu navegador (con export/import de respaldo).

## Desarrollo

```bash
npm install
npm run dev    # http://localhost:5173/flowOS/
npm test       # tests en modo watch
npm run build
```

## Honestidad científica

Esta app no te hace más inteligente (la evidencia de transferencia lejana del
brain-training es débil — Simons et al. 2016). Lo que sí ofrece: un calentamiento
que eleva tu alerta, dopamina a cambio de esfuerzo en vez de scroll pasivo, y
práctica directa del hábito de sostener la atención. El detalle está en la
pantalla "¿Por qué funciona?" y en `docs/superpowers/specs/`.
